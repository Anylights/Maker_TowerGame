# 正交能源塔防策划案

版本：V0.2

## 1. 项目概述

2.5D 正交俯视角塔防 Roguelike。玩家拥有一座**核心能源塔**（唯一核心建筑、资源中枢、失败条件），向周围的**基础塔**供能。基础塔是空白塔体，挂载**圣器**后形成具体功能（攻击 / 收集 / 增幅）。

**总功率恒定**——塔越多则单塔分到越少；远塔本体弱，但传输损耗会变成"地面能源线"对穿过的怪物造成持续伤害。玩家在塔的数量、距离、圣器搭配、资源回收、主动技能之间不断取舍。

单局 20-30 分钟，20 波怪。

## 2. 核心卖点

1. **唯一能源核心**：全局只有一座能源塔，新增基础塔不会增加总功率，只会重新瓜分。
2. **损耗武器化**：传输损耗变成地面能源线伤害——远塔本体弱，但能源线长且伤害高。功率守恒：所有塔功率 + 所有能源线功率 = 总功率。
3. **塔体统一 + 圣器构筑**：基础塔只是插座，圣器决定职业和流派。
4. **战斗与采集同场**：塔可攻击地形物件获取金币 / 材料 / 圣器胚胎。
5. **局外圣器收集 + 牌库出战**：玩家用通关货币购买圣器，每局组牌库出战。

### 3.3 核心圣器（替代旧版"核心模块"）

只能装在能源塔上。**1 个槽位**，Lv.5 解锁，Lv.10 觉醒（效果 +50%）。

核心圣器**单独池子**，不与普通圣器混用。每局 pre-run 选 1 件作为出战核心圣器，Lv.5 自动装备。

首版 6 件：

| 圣器 | 效果 | 流派 |
|---|---|---|
| 稳压圣器 | 距离衰减降低，最低有效供能 25%→35% | 近塔流 |
| 导线圣器 | 能源线转伤 +30%，多线叠加递减减轻 | 远线流 |
| 蓄能圣器 | 主动能量获取 +25%，技能消耗 -10% | 技能流 |
| 集焦圣器 | 距能源塔 ≤2 格的塔功率 +50% | 极近塔流 |
| 充能圣器 | 击杀任何敌人 +1 主动能量 | 大爆发流 |
| 共振圣器 | 同种圣器 ≥3 件时，能源塔总功率 +20% | 主题构筑 |

## 4. 基础塔

| 属性 | 说明 |
|---|---|
| 建造消耗 | `30 + 8 × 已建数量 + 2 × 已建数量²` |
| 圣器槽 | 2 个（主槽 + 副槽，副槽效果 50%-70%） |
| 默认攻击 | 低强度炮弹 |
| 拆除返还 | 战斗中 40%、波间 70%（圣器自动回背包） |
| 暂停供能 | 临时断电某座塔，让其他塔变强 |
| 目标优先级 | 怪物优先 / 地形优先 / 低血优先 / 高甲优先 / 最近能源塔优先 |

基础塔本身不分职业，由圣器决定。

## 5. 能源系统

### 5.1 供能公式

```
P_total = 能源塔总功率
N = 当前被供能的基础塔数量
P_share = P_total / N

d_i = 第 i 座塔到能源塔的距离
R = 能源塔供能半径
attenuation_i = clamp(1 - 0.65 × (d_i / R)^1.35, 0.25, 1.00)

P_delivered_i = P_share × attenuation_i
P_line_i = P_share - P_delivered_i

sum(P_delivered_i) + sum(P_line_i) = P_total   ← 必须守恒
```

### 5.2 功率系数对塔属性的影响

功率主要影响**频率与触发**，不影响单发伤害（避免远塔体感过弱）。

| 属性 | 影响方式 |
|---|---|
| 攻击速度 | 线性，最低保留 30% |
| 炮弹速度 | 轻度，最低 70% |
| 特效触发率 | 线性 |
| 收集效率 | 线性 |
| 增幅强度 | 线性 |
| 主动充能 | 线性 |

### 5.3 能源线伤害

```
line_dps_i = P_line_i × 0.35 × 转伤效率 × 圣器修正
multi_line_decay = [1.00, 0.75, 0.55, 0.40, ...]
```

伤害修正只改变 `P_line_i` 转化为伤害的效率，不改变 `P_line_i` 本身（功率守恒不破坏）。

## 6. 资源系统

| 资源 | 来源 | 用途 |
|---|---|---|
| **金币** | 击杀、收集圣器、攻击地形、波次奖励 | 建塔、能源塔升级、圣器精炼 |
| **材料** | 攻击地形、精英怪、Boss、分解圣器 | 圣器精炼、融合、能源塔升级 |
| **主动能量** | 收集圣器、能源水晶、连击 | 主动技能 |

三者共用 = 经济竞争。建塔、升级、精炼都在抢同一笔钱。

## 7. 圣器系统

### 7.1 三大类

1. **攻击型**：改变炮弹、伤害类型、状态效果
2. **收集型**：提高金币、材料、主动能量获取
3. **增幅型**：增强周围塔、能源线或能源塔属性

### 7.2 槽位

每座基础塔 2 槽：主槽（决定主要功能）+ 副槽（附加效果，强度 50%-70%）。

### 7.3 品质

| 品质 | 颜色 | 定位 |
|---|---|---|
| 普通 | 白 | 基础效果 |
| 稀有 | 蓝 | 明确流派倾向 |
| 史诗 | 紫 | 强联动或机制改变 |
| 传说 | 金 | 围绕它构筑整套打法 |

**核心原则**：稀有度 ≠ 数值倍率。Common 圣器在合适构筑下能打过 Legendary。强圣器必带反噬（如"+50% 攻速但范围 -2"）。**至少 30% 的圣器必须是"改规则"型**，不只是"加数值"。

### 7.4 局内获取

3 个掉落通道：

1. **波次结束 3 选 1 + Skip**（Skip 合法、有小奖励）
2. **精英怪必给 1 件**
3. **场景物件开采**：古代遗迹掉"圣器胚胎"，胚胎孵化为 2 选 1

Boss 必给 1 件 Boss 专属圣器（质变型 + 反噬）。

### 7.5 精炼（局内成长，必成功）

消耗同名碎片 + 材料 + 金币。每 +3 解锁一个小质变（前移让玩家更早感受变化）。

| 等级 | 变化 |
|---|---|
| +1~+3 | 数值提升 + 第一个小质变 |
| +4~+6 | 第二个质变 + 次要词条 |
| +7~+9 | 第三个质变 + 流派联动 |
| +10 | 终极质变 |

### 7.6 融合（局内高风险成长）

选择 2-3 件圣器作为素材。

| 融合类型 | 成功率 |
|---|---|
| 同名低品质融合 | 90% |
| 同类不同名融合 | 70% |
| 跨类融合 | 55% |
| 史诗以上跨类融合 | 40% |
| 传说定向融合 | 25% |

**失败处理**：素材 50% 返还（同等价值碎片）+ 产出一个降级品。**不再纯销毁**。可加"稳定剂"材料提高成功率，但不能达到 100%。

### 7.7 局外购买（核心 meta 循环）

通关获得"星币"（局外货币），按表现给（剩余血量、清波速度、开采率）：通关一次 200-800 星币。

商店价格按品质阶梯：

| 品质 | 价格 |
|---|---|
| 白 | 100 |
| 蓝 | 400 |
| 紫 | 1500 |
| 金 | 5000 |

**全开放可见**，但购买权限受局外能源塔等级限制（见 §10）。

局内掉落按品质权重：白 60% / 蓝 25% / 紫 12% / 金 3%。

### 7.8 牌库（出战配置）

玩家从已购买的圣器中组一个**出战牌库**，局内掉落只从牌库中抽取。牌库可在每局开始前调整。

牌库容量随局外能源塔等级解锁：

| 局外能源塔等级 | 牌库容量 |
|---|---|
| 1 | 6 |
| 2 | 7 |
| 3 | 8 |
| 4 | 9 |
| 5 | 10 |

## 8. 怪物

### 8.1 普通怪（首版 6 种）

| 怪物 | 特点 | 克制 |
|---|---|---|
| 行尸 | 标准 | 任意 |
| 群虫 | 多、低生命 | 范围、燃烧、能源线 |
| 甲壳兽 | 高甲 | 腐蚀、破甲、雷击 |
| 疾行者 | 高速 | 减速、冻结、能源线 |
| 护盾怪 | 能量盾 | 高频攻击、雷鸣 |
| 吞能者 | 经过能源线时降低线伤 | 塔本体输出 |

### 8.2 精英词缀（首版 4 种，可任意叠加到普通怪）

| 词缀 | 效果 |
|---|---|
| 厚甲 | 护甲 +200% |
| 迅捷 | 速度 +50% |
| 抗燃 | 燃烧时间 -70% |
| 吸能 | 经过能源线时回血 |

2 词缀 × 6 普通怪 = 24 种精英变体（用 modifier 系统翻倍敌人多样性）。

### 8.3 Boss

首版 2 个，每个针对核心机制设计：

1. **裂山巨像**：周期获得护甲 + 砸碎资源地形 → 考验破甲与火力分配
2. **吞线母体**：经过能源线时**吸收**线伤获得护盾，并周期"吸取场上 30% 总功率持续 5 秒" → 强迫所有构筑都得为它预留方案

## 9. 场景物件

| 物件 | 掉落 | 邻接 buff |
|---|---|---|
| 石块 | 金币、少量材料 | 无 |
| 矿石 | 材料、主动能量 | 相邻塔挂载收集型时，效果 +20% |
| 古代遗迹 | 圣器胚胎、融合材料 | 相邻塔的圣器触发率 +15% |
| 能源水晶 | 主动能量 | 相邻塔功率系数 +10% |
| 封印山体 | 稀有圣器、塔位 | 击破后开放新建造格 |

**邻接 buff 把采集与构筑耦合**，不只是"金币按钮"。

## 10. 局外系统

### 10.1 局外能源塔等级（1 → 5）

| 局外等级 | 局内开局等级 | 商店解锁 | 牌库容量 | 其他 |
|---|---|---|---|---|
| 1 | Lv.1 | 白 | 6 | — |
| 2 | Lv.2 | 白、蓝 | 7 | 初始金币 +20 |
| 3 | Lv.3 | 白、蓝、紫 | 8 | 初始材料 +20 |
| 4 | Lv.4 | 白、蓝、紫 | 9 | 初始主动能量 +30 |
| 5 | Lv.5 | 白、蓝、紫 | 10 | 解锁挑战难度 |

**金圣器另走"成就解锁"**（如：单局通关 / 用同流派通关 / 击败特定 Boss），不通过等级解锁。这避免老玩家"再升一级就能买金"的胡萝卜钉死。

### 10.2 局外能源塔升级消耗（陡曲线）

| 升级 | 星币 |
|---|---|
| 1→2 | 500（约 1-2 局） |
| 2→3 | 2000 |
| 3→4 | 6000 |
| 4→5 | 15000 |

早爽后稳：玩家很快能体验"开始买蓝了"，但攒到 Lv.3 解锁紫需要明显投入。

### 10.3 货币消费桶

```
[星币]
   ├─ 升级局外能源塔（解锁更高商店权限 + 局内开局加成 + 牌库容量）
   ├─ 购买圣器（按品质阶梯）
   └─ 全部买完后：兑换"开局加成"（额外金币、材料、能量）
```

三者天然取舍：新手优先升能源塔解锁蓝；老手砸圣器和牌库容量。

### 10.4 不做玩家间交易

替代方案（首版可暂缓，但架构预留）：

- **每周种子排行榜**（同种子开局、比 build）
- **异步圣器礼物**（通关后留下 1 件圣器随机送达，不可指定、不可交换、不可买卖）
- **Build 截图分享**（社区自然形成）

|---|---|---|

## 12. 局内流程

20 波 / 单局 20-30 分钟。

| 阶段 | 波次 | 目标 |
|---|---|---|
| 开局 | 1-5 | 学会建塔、供能、基础圣器 |
| 扩张 | 6-10 | 引入地形开采、收集型圣器 |
| 构筑 | 11-15 | 流派成型、精英词缀出现 |
| 压力 | 16-19 | 多路线、混合精英 |
| 决战 | 20 | Boss 检验布局 |

每波前有准备时间。**支持快进（2x / 4x）+ 暂停建造**（参考 BTD6 标配，是 Roguelike TD 的卫生标准）。

## 13. UI 关键要求

### 13.1 主界面常驻

- 能源塔生命、等级
- 当前金币、材料、主动能量
- **当前总功率 / 基础塔有效功率总和 / 能源线功率总和**（功率守恒可视化）
- 当前供能塔数量
- 下一波怪物预告

### 13.2 建造预览

- 是否在供能范围内
- 距能源塔距离（格）
- **建造后该塔预计供能 %**
- **建造后所有塔平均供能变化**
- 能源线路径预览
- 能源线预计伤害

### 13.3 全图功率流热力图（按 Alt 切换）

- 能源塔→基础塔的功率流粗细
- 损耗占比可视化
- 类似 Factorio 电网视图——把"功率守恒"从口号变成可点开验证的事实

### 13.4 能源线表现

- **动态箭头 / 流光显示流向**（学 Mindustry 教训：静态线条会让玩家看不懂流向）
- 损耗越高越亮
- 造成伤害时电弧效果

## 15. 关键平衡原则

1. 总功率从 Lv.1 的 100 → Lv.10 的 200，**翻倍上限**。塔分流压迫感必须保留。
2. 局外能源塔最高 Lv.5，老玩家开局顺，但仍需局内升 5 次才解锁核心圣器系统——保留构筑期。
3. 商店购买权限受局外等级限制，避免新手"先买齐金的"最优解。金圣器靠成就解锁。
4. 强圣器必带反噬。普通圣器在对的构筑下能打过传说。
5. 融合保留风险（不会 100% 成功），但失败不归零。
6. 远塔有价值（线伤）但不能成为唯一最优解（吞能者、抗线敌人制衡）。

## 16. 第一阶段验证目标

1. 玩家是否理解"总功率恒定，塔越多单塔越弱"
2. 玩家是否愿意为了能源线伤害把塔放远
3. 圣器是否真的让同一种基础塔产生不同玩法
4. 攻击地形是否形成有趣的火力取舍
5. 局内 10 级能源塔升级 + 核心圣器是否提供清晰构筑收束
6. 局外购买 + 牌库出战是否提供持续的"我又买到新圣器"动力
7. 精炼和融合是否撑得起单局构筑期待
