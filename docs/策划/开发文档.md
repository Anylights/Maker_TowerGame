## 4. 局外系统详细交互

### 4.1 主菜单

**显示内容**：
- 游戏 logo
- 当前星币（货币）
- 当前局外能源塔等级（Lv.1-5）
- 5 个按钮：开始游戏 / 商店 / 牌库 / 升级能源塔 / 设置

**进入条件**：游戏启动后第一个看到的画面。

### 4.2 商店

**显示内容**：
- 当前星币
- 当前局外能源塔等级 + 解锁的稀有度提示（如 "Lv.2 已解锁：白、蓝"）
- 圣器列表，按品质分组（白 / 蓝 / 紫 / 金）
- 每件圣器卡片：图标、名称、品质边框颜色、价格、效果描述、"购买" 按钮（已购买的显示"已拥有"）
- 金圣器显示锁的图标 + "完成 XX 成就解锁" 提示（成就系统首版可暂用占位文案）

**交互流程**：
1. 玩家点击主菜单"商店"按钮 → 进入商店界面
2. 玩家点击某件圣器的"购买"按钮
   - 检查：星币是否够 + 该稀有度是否解锁 + 玩家是否未拥有
   - 不满足 → 按钮变灰显示禁用原因
   - 满足 → 弹出确认框 → 确认后扣星币、圣器加入收藏
3. 玩家点击"返回"按钮 → 回到主菜单

**价格**（来自 `data/balance.json` 的 `shop_prices`）：白 100 / 蓝 400 / 紫 1500 / 金 5000

**购买权限**（来自 `out_of_run_energy_tower.level_curve` 的 `shop_unlock_rarities`）：
- Lv.1 → 白
- Lv.2 → 白、蓝
- Lv.3-5 → 白、蓝、紫
- 金 → 永远走成就解锁，不靠等级

### 4.3 牌库构建

**显示内容**：
- 当前牌库容量（如 "6/6"，由局外能源塔等级决定）
- 已购买圣器列表（按类型筛选：攻击 / 收集 / 增幅 / 全部）
- 当前牌库（高亮显示）
- 核心圣器选择区：6 个核心圣器图标，玩家选 1 件作为本局核心
- "进入战斗" 按钮

**交互流程**：
1. 玩家点击主菜单"牌库"按钮
2. 看到当前牌库 + 已购买圣器
3. 点击已购买圣器 → 加入牌库（如果牌库已满则提示）
4. 点击牌库中的圣器 → 移出牌库
5. 点击核心圣器图标 → 选中（互斥，只能选 1 件）
6. 点击"进入战斗" → 进入局内
   - 检查：牌库至少 1 件 + 必须选 1 件核心圣器
   - 不满足 → 提示

**牌库容量**（来自 `out_of_run_energy_tower.level_curve.deck_capacity`）：Lv.1=6, Lv.2=7, ... Lv.5=10

### 4.4 局外能源塔升级

**显示内容**：
- 当前局外能源塔等级
- 下一级带来的变化（in_run_start_level、shop_unlock_rarities、deck_capacity、extras）
- 升级费用（星币）
- "升级" 按钮

**交互流程**：
1. 主菜单点击"升级能源塔"
2. 显示当前 Lv.X → Lv.X+1 的对比表
3. 点击"升级" → 检查星币 → 扣费、等级 +1
4. 已 Lv.5 时显示"已满级"

**费用**（来自 `out_of_run_energy_tower.upgrade_costs_starcoins`）：500 / 2000 / 6000 / 15000 星币

### 4.5 持久化数据

需要保存（建议本地存储）：
- `starcoins`：当前星币数
- `owned_artifacts`：已购买的圣器 ID 列表
- `current_deck`：当前牌库（圣器 ID 列表）
- `current_core_artifact`：当前选中的核心圣器 ID
- `out_of_run_energy_tower_level`：1-5
- `unlocked_gold_artifacts`：通过成就解锁的金圣器 ID 列表
- `total_runs`：总通关次数（用于成就系统）

## 5. 局内详细交互

### 5.1 单波次循环

每波分 3 阶段：

```
[准备阶段] (10 秒倒计时, 可被玩家点击"开始")
   - 玩家可建造 / 拆除塔
   - 玩家可装备 / 拆除圣器（免费）
   - 玩家可释放主动技能（如有能量）
   - 玩家可精炼 / 融合圣器
   - 显示下一波怪物预告
   ↓
[战斗阶段] (按 spawn_groups 自动开始)
   - 怪物按配置依次刷出
   - 塔自动攻击
   - 玩家可继续建造 / 装备圣器（战斗中换装有 5 秒冷却）
   - 玩家可释放主动技能
   - 玩家可点击地形 / 怪物指定塔的攻击目标
   ↓
[波次结算] (所有怪物死亡 / 到达终点 / 玩家失败)
   - 给金币奖励（基础 + 表现奖励）
   - 弹出 3 选 1 圣器奖励面板（详见 §5.3）
   - 显示下一波倒计时
```

### 5.2 建造 / 拆除塔

**建造流程**：
1. 玩家点击 HUD 的"建造塔"按钮（或按 B） → 进入建造模式
2. 鼠标悬停在地图格子上：
   - 合法格子（空地 + 在能源塔供能半径内）→ 显示绿色高亮 + 建造预览
   - 非法格子（地形 / 路径 / 能源塔上 / 超出半径）→ 显示红色高亮
3. 建造预览必须显示：
   - 距能源塔距离（格）
   - 建造后该塔预计供能 %
   - 建造后所有塔的平均供能变化（"43% → 38%"）
   - 能源线预览（虚线，能源塔到此塔）
   - 能源线预计 DPS
   - 建造费用（递增公式：`30 + 8N + 2N²`，N = 已建塔数）
4. 玩家点击合法格子 → 扣金币 → 塔建造完成 → 退出建造模式
5. 按 ESC / 右键 → 退出建造模式

**拆除流程**：
1. 玩家点击塔 → 显示塔面板（详见 §7.2）
2. 点击"拆除"按钮 → 弹出确认框（"返还 X 金币，圣器自动回背包"）
3. 确认 → 塔消失，金币 / 圣器返还
   - 战斗中拆除返还 40%
   - 准备阶段拆除返还 70%

**暂停供能**：
1. 在塔面板点击"暂停供能"按钮
2. 该塔变暗（断电视觉），不再攻击 / 产线伤
3. 其他塔功率立即重新分配（变强）
4. 再次点击 → 恢复供能

### 5.3 圣器掉落 3 选 1

**触发时机**：
- 每波次结束 → 1 次掉落
- 精英怪死亡 → 1 次掉落
- 古代遗迹被打破 → 产出"圣器胚胎"，胚胎在波次结算时孵化为 2 选 1（少一个候选）

**显示内容**：
- 弹出面板，正中显示 3 张圣器卡片（带品质边框）
- 每张卡片：图标、名称、品质、效果描述
- 右下角"Skip" 按钮（带"+50 金币"提示）

**抽取逻辑**：
1. 从牌库（玩家局外组的牌库）中按品质权重独立抽 3 候选
   - 权重：白 60% / 蓝 25% / 紫 12% / 金 3%（来自 `data/balance.json` `drop_weights`）
   - 如果牌库中没有该品质的圣器 → 重抽该候选位（不替代）
2. 玩家点击某张卡片 → 进入"装备到哪座塔"模式
3. 玩家点击场上某座塔 → 弹出"装备到主槽 / 副槽"
4. 玩家点击槽位 → 圣器装备成功，效果立即生效
5. 玩家点击"Skip" → 关闭面板，给 50 金币奖励

**装备规则**：
- 每塔 2 槽位（主槽 + 副槽）
- 主槽圣器效果按 100% 计算
- 副槽圣器**数值类**效果按 60% 计算（统一系数）
- 副槽**质变类**效果完全不生效（详见 §6.2）
- **副槽不能装与主槽相同 ID 的圣器**
- 装备时如果该塔槽位已满 → 弹出"替换 / 取消"，替换后旧圣器返回背包

### 5.4 圣器精炼

**触发时机**：准备阶段（战斗中不能精炼）。

**入口**：HUD 左下"圣器背包" → 点击某件圣器 → 详情面板"精炼"按钮。

**显示内容**：
- 当前等级 → 下一等级 的效果对比
- 精炼成本（金币 + 同名碎片）
- "精炼" 按钮（资源不够则禁用 + 显示缺什么）

**交互流程**：
1. 选择圣器 → 点击精炼
2. 显示成本（来自 `data/balance.json` `refining.cost_per_level`）
3. 资源够 → 确认 → 扣资源、圣器升级 +1
4. 升级到 +3、+6、+9、+10 时弹窗提示"解锁质变"

**碎片来源**：
- 重复获得已拥有的圣器时，自动转化为该圣器的碎片
- 分解圣器换碎片（详见 §5.10）

### 5.5 圣器融合

**触发时机**：准备阶段。

**入口**：HUD 左下"融合"按钮，或圣器背包面板内的"融合"标签。

**显示内容**：
- 3 个素材槽 + 1 个产物预览槽
- 拖入素材后：显示成功率 + 可能产物列表
- "稳定剂" 按钮（消耗 X 材料 +15% 成功率）
- "融合" 按钮

**交互流程**：
1. 选 2-3 件圣器拖入素材槽
2. 系统计算融合类型（同名 / 同类不同名 / 跨类 / 跨史诗 / 传说定向），显示成功率
3. 玩家可加稳定剂提高成功率（不能 100%）
4. 点击"融合" → 滚成功率
   - 成功 → 弹窗显示新圣器、加入背包
   - 失败 → 50% 素材返还为碎片 + 1 件降级品（如紫紫融合失败 → 拿到 1 件蓝品质）

**成功率**（来自 `data/balance.json` `fusion.success_rates`）：
- 同名低品质：90%
- 同类不同名：70%
- 跨类：55%
- 跨史诗+：40%
- 传说定向：25%

### 5.6 主动技能释放

**显示内容**：
- HUD 底部 6 个技能图标，按数字键 1-6 对应
- 每个图标显示：技能名、消耗、CD（圆形遮罩剩余时间）
- 主动能量条：当前 / 上限

**交互流程**：
1. 玩家按数字键 1-6（或点击图标）
2. 检查能量 + CD
   - 能量不足 → 屏幕震动 + UI 提示"能量不足"
   - CD 中 → 提示"冷却中"
3. 满足 → 触发释放
   - **立即生效型**（超频、能源线过载、紧急护盾、塔体重启、全域回收）：扣能量、开始 CD、效果立刻施加
   - **需选目标型**（天降雷电）：进入"选目标"模式，鼠标变成准星，点击地图位置释放
4. 释放成功 → 系统通知"主动技能已释放"事件（用于跨源圣器联动，如过载继电器、共振触发、注能弹药等）

**6 个技能数值**：见 `data/skills.json`

### 5.7 攻击地形物件

**地图上有 5 种地形物件**（来自 `data/balance.json` `scene_objects`）：
- 石块、矿石、古代遗迹、能源水晶、封印山体

**交互流程**（两种方式）：

方式 A — 直接指派：
1. 玩家点击地形物件 → 弹出选项："指派塔攻击 / 取消"
2. 点击"指派塔攻击" → 进入"选塔"模式 → 点击附近塔
3. 该塔目标优先级临时切换为攻击此物件，物件破坏后恢复

方式 B — 设置默认优先级：
1. 玩家点击塔 → 塔面板 → 目标优先级下拉选"地形优先"
2. 塔自动攻击射程内最近的地形物件，没有时再打怪

**物件破坏时**：
- 销毁该物件
- 掉落资源（金币 / 材料 / 主动能量 / 圣器胚胎，按物件类型）
- 该物件提供的"邻接 buff"失效

**邻接 buff**：
- 矿石：相邻塔挂载收集型时效果 +20%
- 古代遗迹：相邻塔的圣器触发率 +15%
- 能源水晶：相邻塔功率系数 +10%
- 石块、封印山体：无 buff（封印山体破坏后开放新建造格）

### 5.8 暂停 / 加速 / 退出

**暂停**：按 Space → 游戏暂停 + 显示暂停菜单（继续 / 设置 / 放弃当局）

**加速**：HUD 右上角速度按钮（1x / 2x / 4x），或按 Tab 切换

**放弃当局**：暂停菜单点击 → 弹出确认框 → 确认后视为失败、按当前波次给少量星币、返回主菜单

### 5.9 结算屏

**触发**：能源塔死亡（失败）/ Wave 20 完成（通关）/ 玩家放弃

**显示**：
- 结果（"通关" / "失败 Wave X" / "放弃"）
- 最高 Wave、用时、击杀数
- 获得星币（基础 + 表现奖励 + Boss 击杀奖励）
- "返回主菜单" 按钮

**星币奖励公式**（来自 `data/balance.json` `starcoin_rewards`）：
- 基础 200
- 每剩余 1% 能源塔 HP +5
- 每提前完成的波次 +10
- Boss 击杀 +100
- 上限 800

## 6. 系统细节

### 6.1 能源系统计算时机

每次发生以下事件时，**全场重算所有塔的功率分配**：
- 建造塔 / 拆除塔
- 暂停 / 恢复供能
- 能源塔升级
- 装备 / 拆除影响功率公式的圣器（稳压、网络、远程压缩等）
- 主动技能"超频"释放或结束（影响 P_total）
- Boss 吞线母体的"吸取功率"开始或结束

**算法**（详见 [策划案 V0.2 §5.1](正交能源塔防策划案_V0.2.md)）：

```
P_total = 能源塔总功率（按等级，含临时 buff/debuff）
N = 当前被供能的基础塔数量（不含暂停供能的）

for 每座塔 i：
  P_share = P_total / N
  d = distance(塔 i, 能源塔)
  R = 能源塔供能半径
  attenuation = clamp(1 - 0.65 × (d / R)^1.35, min_attenuation, 1.00)
  P_delivered_i = P_share × attenuation
  P_line_i = P_share - P_delivered_i

power_ratio_i = P_delivered_i / P_total  // 用于 UI 显示供能 %
```

**守恒约束**（开发期间用断言验证）：
```
sum(P_delivered_i) + sum(P_line_i) === P_total
```

例外：网络圣器的次级线**显式**不计入守恒。

### 6.2 主槽 vs 副槽规则

每塔 2 槽：主槽 + 副槽。

| Effect 类型 | 主槽 | 副槽 |
|---|---|---|
| StatModifier（数值类） | 100% | 60% |
| OnHitStatus（状态附加） | 100% | 60% 触发率 |
| EnergyLineModifier | 100% | 60% |
| SpawnSecondaryLines（网络圣器） | 生效 | **不生效** |
| TransformBullet（棱镜、高爆） | 生效 | **不生效** |
| CustomLogic：tower_disable_attack（凝聚塔、吞噬线、防御阵地塔） | 生效 | **不生效** |
| CustomLogic：expand_slots（总管塔） | 生效 | **不生效** |
| CustomLogic：其他改规则类（网络圣器的次级线、元素反应等） | 生效 | **不生效** |
| SkillSynergy | 100% | 60% |

**约束**：
- 主副槽不能装相同 ID 的圣器
- 战斗中换装有 5 秒冷却（准备阶段免费）

### 6.3 状态效果叠加规则

| 状态 | 叠加规则 |
|---|---|
| **燃烧 (burn)** | 同一怪物身上保留 max(existing_DPS, new_DPS) 和 max(existing_remaining_duration, new_duration) |
| **冰冻 (freeze)** | 按层数累加，最多 5 层；满 5 层冻结 1.5s 后清空层数；不冻结时每秒衰减 0.5 层 |
| **感电 (shock)** | 不在怪物身上叠加，是命中事件触发的连锁；雷鸣圣器的连锁与天降雷电的连锁互不影响 |
| **腐蚀 (corrode)** | 按层数累加，最多 8 层；护甲减少 = 0.25 + 0.05 × (stacks - 1)，cap 0.60；持续 8 秒，最后一次命中刷新持续时间 |

### 6.4 子弹形式与效果兼容矩阵

每座塔只能有 1 种子弹形式（炮弹默认 / 棱镜→激光 / 高爆→范围）。圣器的 `bullet_form_compatibility` 字段定义兼容关系。

| 子弹形式 | 兼容效果圣器 | 不兼容（灰显禁装） |
|---|---|---|
| 炮弹 | 火种、冰晶、雷鸣、裂片、腐蚀 | — |
| 激光 | 冰晶、雷鸣（穿透型） | 火种、裂片、高爆、腐蚀 |
| 范围 | 火种、腐蚀（区域型） | 冰晶、雷鸣、裂片、棱镜 |

**UI 必须**：装备界面对不兼容的圣器显示灰色 + 鼠标悬停提示原因。

### 6.5 主动技能释放后的影响（跨源联动）

释放任意技能时通知系统"skill.cast"。订阅此事件的圣器：

| 圣器 | 响应 |
|---|---|
| 过载继电器 | 所有线伤 +150% × 5s |
| 共振触发 | 装备此圣器的塔下一发 ×3 + 穿透 |
| 注能弹药 | 所有塔攻速 +100% × 5s |
| 防御阵地塔 | 自身护盾刷新到 100 |

这些 buff 同时存在时**叠加生效**（不互斥）。

### 6.6 怪物特殊行为

| 怪物 | 行为细节 |
|---|---|
| **吞能者** | 经过能源线时，该线对**自己**的伤害降为 0；对其他怪物不影响 |
| **护盾怪** | 盾 HP 与本体 HP 分离；盾未破时免疫所有 status；盾破后变成普通行尸 |
| **群虫** | 一个 spawn_group 刷量自动 ×2（waves.json 中 count 字段是基础值） |
| **裂山巨像（Boss）** | 每 15s 获得护甲 +100%（持续 8s）+ 视觉发红光；移动到地形物件上时砸碎该物件 |
| **吞线母体（Boss）** | 经过能源线时该线对它伤害为 0，并加 100 HP 护盾（每条线给一次）；每 30s 吸取场上 30% 总功率持续 5s（HUD 数字下降 + 视觉变暗） |

### 6.7 圣器掉落池规则

局内掉落池 = 玩家局外组的牌库。
- 牌库 6-10 件
- 每次掉落滚 3 候选，按品质权重独立抽
- 如果牌库内某品质数量不足 → 重抽该候选位
- 玩家选 1 件 + 装备到塔，或 Skip
- 同一件圣器可以被同一局多次抽到（用作精炼碎片或装备到不同塔）

### 6.8 局外能源塔等级对局内的影响

| 局外等级 | 局内开局能源塔等级 | 商店解锁稀有度 | 牌库容量 | 其他开局加成 |
|---|---|---|---|---|
| 1 | Lv.1 | 白 | 6 | — |
| 2 | Lv.2 | 白、蓝 | 7 | 初始金币 +20 |
| 3 | Lv.3 | 白、蓝、紫 | 8 | 初始材料 +20 |
| 4 | Lv.4 | 白、蓝、紫 | 9 | 初始主动能量 +30 |
| 5 | Lv.5 | 白、蓝、紫 | 10 | 解锁挑战难度 |

注意：局外 Lv.5 = 局内开局 Lv.5（已解锁核心圣器槽），Lv.10 觉醒仍需局内升 5 次。

### 6.9 核心圣器装备规则

- pre-run 玩家在牌库构建界面选 1 件核心圣器
- 局内能源塔升到 Lv.5 时**自动装备**该核心圣器
- Lv.10 时**自动觉醒**（应用 lv10_awakening 效果，叠加在主效果之上）
- 局内不可更换核心圣器

## 8. VFX 视觉需求

VFX 是这游戏的"灵魂"——能源系统抽象，玩家必须靠视觉理解功率分配。**任何关键级 VFX 缺失会直接影响玩家是否能 get 到核心机制**。

本节只描述视觉表现，不涉及实现技术。

### 8.1 关键级（必做，缺了游戏不成立）

| VFX | 视觉描述 |
|---|---|
| **能源线流向** | 从能源塔到基础塔的能量流动光带，必须看出方向（粒子从能源塔向基础塔流动，每秒 1-2 个） |
| **能源线损耗强度** | 损耗越高，线越亮、越粗、颜色越偏暖（白蓝 → 金白 → 金红） |
| **能源线伤害命中** | 怪物站在线上时，电弧从线打到怪物（每 0.2s 一闪） |
| **塔功率系数** | 高功率塔更亮 + 微微脉冲；低功率塔暗淡 + 发射间隔明显变长 |
| **能源塔核心** | 中央旋转的核心，供能紧张时跳动加快（频率与 N / 供能率挂钩） |
| **建造预览** | 半透明塔 sprite + 半透明供能范围圆 + 虚线能源线 |
| **网络圣器次级线** | 次级线必须**视觉区别于**主线（更细、更暗、流动方向相反或虚线） |
| **暂停供能塔** | 塔变灰 + 顶部图标显示电源关闭符号 + 该塔的能源线消失 |

### 8.2 重要级（强烈推荐）

| VFX | 视觉描述 |
|---|---|
| **子弹形式区分** | 炮弹（飞行体 + 短尾迹） / 激光（持续光束） / 范围（弹道 + 命中爆炸） 视觉差异显著 |
| **状态效果叠加** | 燃烧（橙色火焰 + 烟雾） / 冰冻（蓝色冰晶覆盖 + 数字层数） / 感电（黄色电弧脉冲） / 腐蚀（绿色滴液） |
| **元素反应** | 蒸发（红 + 蓝 → 白色爆炸 + 字"VAPORIZE"） / 麻痹（黄 + 蓝交叉电弧） / 过载（紫色殉爆波） / 侵蚀（绿色光晕） |
| **主动技能** | 超频（能源塔外圈展开 + 地面符文亮起） / 雷电（天空闪光 → 落雷柱） / 能源线过载（所有线脉冲式爆闪） / 紧急护盾（六边形光罩） / 全域回收（金币飞向能源塔） / 塔体重启（所有塔短暂变白闪烁） |
| **共振触发** | 释放技能瞬间，装备此圣器的塔头顶弹出"×3"数字 + 弹道发光 |
| **塔受到伤害** | 红色闪烁 + HP 条 |
| **怪物死亡** | 粒子 burst + 金币飞起动画 |
| **超频期间** | 能源塔半径外圈展开光环 + 所有塔有金色边框 |

### 8.3 加分级（首版可选）

| VFX | 视觉描述 |
|---|---|
| **全图功率热力图**（按 Alt 键切换） | 能源塔→基础塔的功率流粗细可视化 |
| **圣器装备瞬间** | 塔头顶圣器图标短暂放大 + 光环 |
| **波次切换** | 屏幕轻微颤动 + "Wave N" 文字飞入 |
| **Boss 入场** | 屏幕震动 + 警告条 + Boss 名字字幕 |
| **危机时刻** | 能源塔 HP < 30% 时屏幕边缘红色脉冲 |
| **金币飞入** | 拾取时金币图标飞向 HUD |

## 11. 关键测试场景

实现到对应阶段时用以下场景验证。

### 11.1 能源系统
1. 1 塔距能源塔 1 格 → 供能 ≈ 95%
2. 1 塔距能源塔 7 格 → 供能 ≈ 25%（最低）
3. 双塔（一近一远）→ 总功率守恒
4. 8 塔密集 → 每塔供能 < 30% 但攻速仍 ≥ 30%（floor 生效）
5. 能源塔升 Lv.5 → 所有塔重新分配
6. 暂停 1 塔供能 → 其他塔功率立刻提升

### 11.2 战斗循环
1. 1 塔默认 → Wave 1 击杀单只行尸 5 ± 1 秒
2. 怪物穿过 1 条能源线持续 3s → 受伤可见
3. 怪物到达能源塔 → 能源塔 -1 HP

### 11.3 圣器（每件单独验证）
1. 装火种 → 命中后 4s 内每秒怪物 -2 HP
2. 装棱镜 → 子弹变激光 + 穿透
3. 装高爆 → 命中后 1.5 格内全部受伤
4. 装冰晶 → 5 次命中后冻结 1.5s
5. 装雷鸣 → 命中跳跃 2 次
6. 装裂片 → 命中后 4 个小弹片
7. 装腐蚀 → 5 次命中后怪物护甲 -45%
8. 装网络圣器 → 此塔 3 格内自动出现次级线（视觉可见，且不互连）
9. 守恒断言不报错（次级线不计入）
10. 装吞噬线 → 此塔不发射，能源线伤害 ×2.5
11. 装借力圣器 → 5 格内有塔时此塔伤害提升
12. 装总管塔 + 距其他塔 ≥3 格 → 槽数 +2
13. 装注能弹药 + 释放任意技能 → 5s 内所有塔攻速 +100%
14. 装蓄力击 + 命中 3 次 → +1 主动能量
15. 装元素反应 + 怪物身上有燃烧 + 冰冻 → 触发蒸发爆炸
16. 装稳压圣器 → 远塔最低有效供能 25% → 40%
17. 装集焦圣器 → 距能源塔 ≤2 格的塔功率系数 +50%

### 11.4 主动技能
1. 按 1 释放超频 → 总功率 +50% × 8s + 视觉变化
2. 按 2 释放天降雷电 → 鼠标变准星 → 点击地图 → 800 单体伤害（AoE）
3. 能量不足 / CD 中 → 无法释放，UI 提示

### 11.5 Boss
1. 裂山巨像 → 每 15s 获得护甲 + 视觉发红
2. 裂山巨像移到地形 → 砸碎该物件
3. 吞线母体 → 经过能源线时该线对它伤害为 0 + 获得 100 HP 护盾
4. 吞线母体 → 每 30s 总功率 -30% × 5s（HUD 数字下降）

### 11.6 局外系统
1. 通关一次 → 获得 ~500 星币
2. 商店买 1 白圣器（100 星币）
3. 牌库放入 6 件圣器 + 选 1 核心圣器
4. 进入下一局 → 掉落只来自牌库
5. 升级局外能源塔 Lv.2 → 解锁蓝圣器购买 + 牌库容量 +1

### 11.7 副槽规则
1. 装火种到副槽 → 燃烧 DPS 是主槽的 60%
2. 装棱镜到副槽 → 子弹**不变**激光（质变类副槽不生效）
3. 主副槽装相同 ID → UI 禁止
