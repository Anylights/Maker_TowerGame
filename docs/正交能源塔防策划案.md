# 正交能源塔防策划案

版本：V0.1 第一版讨论稿

## 1. 项目概述

这是一款 2.5D 正交俯视角塔防游戏。玩家只有一座核心能源塔，能源塔持续输出恒定总功率，并向周围范围内的基础塔供能。基础塔本身是统一的空白塔体，默认可以发射基础炮弹，但真正的战斗定位来自挂载的圣器。玩家通过放置基础塔、分配有限能源、收集和改造圣器，在一波波怪物进攻中保护能源塔。

游戏的核心体验不是“哪里能放塔就放满”，而是“能源永远不够，塔越多越弱，塔越远越亏，但能源线也会变成武器”。玩家需要在塔的数量、距离、圣器搭配、资源回收、主动技能之间不断取舍。

## 2. 核心卖点

1. **唯一能源核心**
   - 全局只有一座能源塔，是基地、资源中枢、主动技能来源，也是失败条件。
   - 所有基础塔共享能源塔的恒定输出，新增塔不会增加总功率，只会重新瓜分功率。

2. **能源分配即战术布局**
   - 基础塔距离能源塔越远，获得的有效能量越少。
   - 衰减掉的能量以“地面能源线”的形式存在，能源线会对穿过的怪物造成持续伤害。
   - 远塔虽然单塔输出低，但会制造更长、更有价值的能源线。

3. **塔体统一，圣器定义流派**
   - 基础塔只是承载体，挂载攻击型、收集型、增幅型圣器后才形成具体角色。
   - 同一座塔可以因为圣器组合变成火炮、冰塔、金币收集塔、能源回收塔、增幅节点或混合功能塔。

4. **战斗、采集、养成同场发生**
   - 玩家既要击败怪物，也要命令塔攻击地图上的山、石头、遗迹等物件。
   - 地形物件掉落金币、材料、圣器胚胎和融合素材。
   - 是否把火力用于清怪，还是用于开采资源，是每波之间和战斗中的关键决策。

5. **高风险圣器融合**
   - 圣器可以精炼升级，也可以融合成新圣器。
   - 融合有失败概率，失败后参与融合的圣器销毁，形成明确的风险收益曲线。

## 3. 目标体验

玩家每一局都应该经历以下节奏：

1. **开局规划**
   - 观察能源塔位置、怪物路径、资源地形和建造格。
   - 决定第一圈基础塔的方向：稳清怪、快采集、早增幅或冒险拉远线。

2. **中期扩张**
   - 建造更多基础塔后，总功率被稀释，旧塔输出下降。
   - 玩家开始围绕“短距离强塔”和“长距离能源线”做布局。
   - 圣器组合逐渐形成流派，例如燃烧连锁、冰冻破甲、金币滚雪球、能源技能流。

3. **后期构筑成型**
   - 玩家有数个关键圣器和主动技能组合。
   - 怪物强度逼迫玩家调整目标优先级、卖塔、换圣器、短期超频。
   - 最终胜利来自能源效率、圣器联动和场地经营，而不是单纯堆数值。

## 4. 基础世界规则

### 4.1 地图视角

- 视角：2.5D 正交俯视角。
- 地图：格子制或半格子制，建议第一版使用格子制，便于能源距离、放置合法性和路径表现。
- 地图层级：
  - 地面层：路径、建造格、资源物件、能源线。
  - 单位层：怪物、塔、投射物、掉落物。
  - 效果层：范围、光效、弹道、状态标记。

### 4.2 能源塔

能源塔是玩家唯一核心建筑。

基础属性建议：

| 属性 | 说明 | 首版建议 |
| --- | --- | --- |
| 生命值 | 归零则失败 | 100 |
| 总功率 | 每秒恒定输出 | 100 功率/秒 |
| 供能半径 | 可供能的最大距离 | 7 格 |
| 主动能量槽 | 释放主动技能的资源池 | 0/100 |
| 默认防御 | 能源塔自身弱攻击或护盾 | 低强度，用于兜底 |

能源塔不建议在首版中允许建造第二座。后续可以做“临时能源锚点”或“能源中继圣器”，但它们只改变供能路径，不产生新的永久总功率。

### 4.3 能源塔升级

能源塔可以在局内升级。升级本身不做复杂分支，只平均提升能源塔的基础数值，让玩家能直观看到“能源核心变强了”。核心模块不放在普通升级过程中，而是在能源塔升到满级后作为一次额外的局内选择出现。

局内能源塔等级建议最高为 5 级：

| 等级 | 总功率 | 生命值 | 供能半径 | 主动能量上限 | 能源线转伤效率 | 额外效果 |
| --- | --- | --- | --- | --- | --- | --- |
| Lv.1 | 100 | 100 | 7 格 | 100 | 100% | 基础状态 |
| Lv.2 | 112 | 115 | 7 格 | 115 | 108% | 无 |
| Lv.3 | 124 | 130 | 7 格 | 130 | 116% | 无 |
| Lv.4 | 136 | 150 | 8 格 | 145 | 124% | 无 |
| Lv.5 | 150 | 175 | 8 格 | 160 | 135% | 解锁核心模块选择 |

升级消耗建议：

| 升级 | 金币 | 材料 |
| --- | --- | --- |
| Lv.1 -> Lv.2 | 80 | 20 |
| Lv.2 -> Lv.3 | 140 | 45 |
| Lv.3 -> Lv.4 | 220 | 80 |
| Lv.4 -> Lv.5 | 320 | 130 |

升级设计原则：

- 升级只提升基础数值，不改变供能公式。
- 升级后重新计算所有基础塔的分功率、距离衰减和能源线功率。
- 总功率提升会让所有塔和能源线一起变强，但塔数量越多单塔仍然会变弱。
- 半径只在关键等级提升，避免每级都扩大地图控制范围。
- 能源线转伤效率是全局倍率，只影响能源线功率转化为伤害的效率，不改变能源线本身分到的功率。
- 升级消耗金币和材料，使能源塔升级、基础塔扩张、圣器精炼之间形成资源竞争。

### 4.4 核心模块

核心模块是局内机制，不是普通升级分支。玩家在本局内把能源塔升到 Lv.5 后，可以从 3 个核心模块中选择 1 个。选择后本局不可更换。

| 模块 | 效果 | 对应流派 |
| --- | --- | --- |
| 稳压模块 | 所有基础塔距离衰减降低，最低有效供能从 25% 提高到 35% | 近塔高效流、稳定炮塔流 |
| 导线模块 | 能源线转伤效率额外 +30%，多条能源线叠加递减减轻 | 远线切割流 |
| 蓄能模块 | 主动能量获取 +25%，主动技能消耗 -10% | 主动技能流 |

核心模块的目标是让满级能源塔成为构筑收束点。普通升级提供稳定成长，核心模块提供本局打法方向。

### 4.5 基础塔

基础塔是所有功能塔的统一塔体。

基础属性建议：

| 属性 | 说明 |
| --- | --- |
| 建造消耗 | 金币，随已建塔数量递增 |
| 能源需求 | 不单独消耗固定值，而是参与瓜分能源塔总功率 |
| 默认攻击 | 低强度炮弹，用于无圣器时也能互动 |
| 圣器槽 | 首版建议 2 个槽位：1 个主槽，1 个副槽 |
| 可拆除 | 返还部分金币，不返还战斗中获得的圣器成长 |
| 可换圣器 | 波间免费换，战斗中换需要短暂冷却或消耗 |

基础塔本身不分炮塔、冰塔、毒塔等职业。职业由圣器决定，这样能突出“圣器构筑”而不是传统塔防的固定塔种。

## 5. 能源系统

能源系统是游戏的核心。

### 5.1 设计目标

1. 总功率恒定，确保玩家不能靠堆塔无限成长。
2. 距离越远，有效供能越低，体现能源衰减。
3. 基础塔功率和能源线功率共同瓜分能源塔总功率，二者相加必须守恒。
4. 衰减能量不是浪费，而是转化为地面能源线功率，再由能源线转伤效率变成伤害。
5. 塔数量越多，单塔功率越少，玩家必须选择重点塔。
6. UI 必须清晰展示每座塔的功率系数，避免玩家感觉数值不透明。

### 5.2 供能公式

首版建议使用简单可读的公式：

```text
P_total = 能源塔总功率
N = 当前被供能的基础塔数量
P_share = P_total / N

d_i = 第 i 座塔到能源塔的距离
R = 能源塔供能半径
attenuation_i = clamp(1 - 0.65 * (d_i / R)^1.35, 0.25, 1.00)

P_delivered_i = P_share * attenuation_i
P_line_i = P_share - P_delivered_i

P_tower_total = sum(P_delivered_i)
P_line_total = sum(P_line_i)
P_tower_total + P_line_total = P_total
```

解释：

- `P_share` 表示每座塔先平均分到的理论功率。
- `attenuation_i` 表示距离衰减，近塔接近 100%，远塔最低保留 25%。
- `P_delivered_i` 是基础塔实际得到的有效功率。
- `P_line_i` 是能源线承载的功率，也就是距离衰减后没有进入基础塔的部分。
- 所有基础塔有效功率 `P_tower_total` 加上所有能源线功率 `P_line_total`，必须始终等于当前能源塔总功率 `P_total`。
- 能源线的各种伤害加成只改变 `P_line_i` 转化为伤害的效率，不额外创造功率。

### 5.3 功率对塔属性的影响

每座塔有一个功率系数：

```text
power_ratio_i = P_delivered_i / P_share_when_only_one_tower
```

其中 `P_share_when_only_one_tower` 可以先取能源塔总功率作为基准。首版为了易懂，可以直接显示为百分比，例如“当前供能：43%”。

功率系数影响：

| 属性 | 建议影响方式 |
| --- | --- |
| 攻击速度 | 与功率系数线性相关，最低保留 30% |
| 炮弹速度 | 轻度相关，最低保留 70% |
| 特效触发率 | 与功率系数相关，避免低能塔触发过强效果 |
| 收集效率 | 与功率系数线性相关 |
| 增幅强度 | 与功率系数线性相关 |
| 主动充能 | 与功率系数线性相关 |

不建议让单发伤害完全随功率线性下降，否则远塔体感会太弱。更好的方式是让功率主要影响频率、触发率和功能效率。

### 5.4 能源线

能源线是能源塔到基础塔之间的一条地面连接线。

规则建议：

- 每座被供能的塔都会生成一条能源线。
- 能源线沿格子中心或最短直线路径绘制。
- 怪物接触能源线时受到持续能量伤害。
- 能源线伤害来自 `P_line_i`，距离越远通常衰减越多，线伤越强。
- 同一怪物同时接触多条能源线时，伤害可叠加，但有递减系数，避免密集线条秒杀。

首版伤害公式：

```text
line_damage_efficiency = 0.35 * 能源线转伤效率 * 圣器/技能修正
line_dps_i = P_line_i * line_damage_efficiency
multi_line_decay = 1.00, 0.75, 0.55, 0.40...
```

这里的 `line_damage_efficiency` 只影响伤害，不影响功率守恒。即使玩家获得能源线伤害加成，`P_line_i` 仍然来自 `P_share - P_delivered_i`，所有基础塔功率和所有能源线功率相加仍然等于 `P_total`。

例子：

- 近塔拿到的有效功率高，塔本体强，但能源线短且伤害低。
- 远塔拿到的有效功率低，塔本体弱，但能源线长且伤害高。
- 玩家可以故意在怪物路径另一侧建远塔，让能源线穿过路径，形成“地面切割网”。

### 5.5 能源拥挤感

当玩家建塔过多时，所有塔都变弱。为了让这种变化有可感知的反馈：

- 建造预览显示“建造后平均供能将从 50% 降至 42%”。
- 塔上方用短暂闪烁显示供能变化。
- 能源塔界面显示当前供能塔数量、总功率、有效功率、线损功率。
- 能源线颜色随线损功率增强，从淡蓝到白蓝再到金白。

## 6. 资源系统

首版建议使用三类资源。

### 6.1 金币

用途：

- 建造基础塔。
- 拆除和重建的经济成本。
- 局内升级能源塔。
- 部分圣器精炼费用。

来源：

- 击杀怪物。
- 收集型圣器拾取掉落金币。
- 攻击地图资源物件。
- 波次奖励。

金币的设计目标是支持局内扩张。

### 6.2 材料

用途：

- 圣器精炼。
- 圣器融合。
- 局内升级能源塔。
- 解锁某些高级主动技能或塔位操作。

来源：

- 山、石头、遗迹、矿脉等场景物件。
- 精英怪和 Boss。
- 分解不需要的圣器。

材料的设计目标是推动玩家攻击地形，而不是只盯怪物。

### 6.3 主动能量

用途：

- 能源塔超频。
- 天降雷电。
- 全场吸取金币。
- 能源线过载。
- 紧急护盾。

来源：

- 收集型圣器转化怪物死亡能量。
- 特定能源收集塔从能源线溢出中回收。
- 地图能源水晶。
- 连击奖励。

主动能量的设计目标是形成战斗中的“爆发按钮”，让玩家有救场与规划空间。

## 7. 圣器系统

### 7.1 圣器定位

圣器是本游戏的主要构筑系统。基础塔只是插座，圣器决定塔的功能、流派和成长路线。

圣器分为三大类：

1. 攻击型：改变炮弹、伤害类型、状态效果和连锁方式。
2. 收集型：提高金币、材料、主动能量获取效率。
3. 增幅型：增强周围塔、能源线或能源塔的部分属性。

### 7.2 槽位规则

首版建议每座基础塔有 2 个圣器槽：

| 槽位 | 作用 |
| --- | --- |
| 主槽 | 决定塔的主要功能，攻击型、收集型、增幅型都可以放 |
| 副槽 | 提供附加效果，强度为主槽的 50% 到 70% |

这样可以形成混合塔：

- 火焰主槽 + 金币副槽：燃烧击杀后额外掉钱。
- 冰霜主槽 + 破甲副槽：冻结后提高物理伤害。
- 收集主槽 + 增幅副槽：经济塔同时给附近塔少量攻速。
- 增幅主槽 + 闪电子槽：辅助塔也能补一点伤害。

后续版本可以解锁第三槽，但首版建议保持 2 槽，降低理解成本。

### 7.3 攻击型圣器

攻击型圣器让基础炮弹获得新的攻击方式。

| 圣器 | 核心效果 | 配合方向 |
| --- | --- | --- |
| 火种圣器 | 炮弹附加燃烧，持续伤害 | 与风、油、爆裂联动 |
| 冰晶圣器 | 减速，叠满后短暂冻结 | 与破甲、高爆联动 |
| 雷鸣圣器 | 命中后跳跃到附近敌人 | 与潮湿、能源过载联动 |
| 裂片圣器 | 炮弹命中后分裂小弹片 | 与暴击、击退联动 |
| 腐蚀圣器 | 降低护甲，造成酸蚀伤害 | 与高速、多段联动 |
| 棱镜圣器 | 炮弹变成穿透光束 | 与直线路径、增幅塔联动 |
| 震荡圣器 | 命中后击退并打断冲锋 | 与能源线、地形杀联动 |
| 标记圣器 | 给敌人施加标记，被攻击时追加伤害 | 与群攻和连锁联动 |
| 高爆圣器 | 炮弹范围爆炸，但射速下降 | 与冻结、聚怪联动 |
| 引力圣器 | 小范围牵引敌人 | 与能源线切割、范围伤害联动 |

攻击流派例子：

- **燃烧扩散流**：火种 + 风蚀 + 高爆，让燃烧在怪群中扩散。
- **冻结爆破流**：冰晶 + 高爆 + 破甲，先控制再打爆发。
- **雷链技能流**：雷鸣 + 能源回收 + 能源塔雷电主动，靠连锁击杀充能。
- **能源线切割流**：震荡 + 引力 + 远距离布塔，把怪推回能源线区域。

### 7.4 收集型圣器

收集型圣器让塔承担经济和主动能量职责。

| 圣器 | 核心效果 | 风险与限制 |
| --- | --- | --- |
| 磁币圣器 | 自动吸取一定范围内金币 | 塔越远供能越低，吸取频率越慢 |
| 掘金圣器 | 攻击地形时额外掉金币 | 清怪能力弱 |
| 采晶圣器 | 攻击矿石时获得主动能量 | 需要玩家分配攻击目标 |
| 灵尘圣器 | 怪物死亡时概率转化材料 | 对 Boss 和精英收益高 |
| 回流圣器 | 回收能源线造成伤害的一部分为主动能量 | 适合远线布局 |
| 分解圣器 | 自动分解低品质掉落，换材料 | 牺牲圣器获取数量 |
| 赏金圣器 | 对被标记敌人提高金币掉落 | 需要标记流配合 |
| 丰收圣器 | 波次结束时按周围塔数量给金币 | 鼓励密集阵地 |

收集型圣器必须有战斗机会成本。如果经济塔完全无损地赚钱，玩家会无脑堆经济；因此收集型塔应当吃供能、占塔位、降低当前火力。

### 7.5 增幅型圣器

增幅型圣器强化周围基础塔或能源系统。

| 圣器 | 核心效果 | 配合方向 |
| --- | --- | --- |
| 鼓舞圣器 | 提高周围塔攻击速度 | 适合近塔集群 |
| 稳压圣器 | 降低周围塔受数量分流的惩罚 | 稀有，不能叠太强 |
| 导线圣器 | 提高附近能源线转伤效率 | 适合远线切割流 |
| 聚焦圣器 | 提高周围塔暴击率或特效触发率 | 适合多段攻击 |
| 透镜圣器 | 增加直线穿透和射程 | 适合棱镜、炮弹流 |
| 充能圣器 | 周围塔命中时额外产生主动能量 | 适合技能流 |
| 维修圣器 | 周围建筑缓慢恢复生命 | 适合防守压力大的关卡 |
| 共鸣圣器 | 周围同类型圣器数量越多，效果越强 | 鼓励主题构筑 |

增幅型圣器建议设置“同类增幅递减”，避免玩家把所有塔做成增幅塔后出现数值失控。

## 8. 圣器成长

### 8.1 品质

首版建议品质分为 4 档：

| 品质 | 颜色 | 定位 |
| --- | --- | --- |
| 普通 | 白 | 基础效果，容易获得 |
| 稀有 | 蓝 | 有明确流派倾向 |
| 史诗 | 紫 | 具备强联动或机制改变 |
| 传说 | 金 | 能围绕它构筑整套打法 |

品质影响初始数值和特殊词条数量，但不应让普通圣器完全没用。普通圣器可以作为精炼和融合材料。

### 8.2 精炼

精炼用于稳定提升已有圣器。

规则建议：

- 消耗同名圣器碎片、材料和金币。
- 每次精炼必定成功。
- 精炼提高基础数值，例如伤害、范围、触发率、收集效率。
- 每 3 级解锁一个小词条强化。

精炼等级建议：

| 等级 | 变化 |
| --- | --- |
| +1 到 +3 | 数值提高 |
| +4 到 +6 | 增加一个次要词条 |
| +7 到 +9 | 强化流派联动 |
| +10 | 获得质变效果 |

### 8.3 融合

融合用于获得新圣器，是高风险成长路径。

基础规则：

- 选择 2 到 3 件圣器作为素材。
- 系统根据圣器类别、品质、等级和流派相性生成融合候选。
- 融合成功获得新圣器。
- 融合失败则参与融合的圣器销毁。

成功率建议：

| 融合类型 | 成功率 |
| --- | --- |
| 同名低品质融合 | 90% |
| 同类不同名融合 | 70% |
| 跨类融合 | 55% |
| 史诗以上跨类融合 | 40% |
| 传说定向融合 | 25% |

可加入“稳定剂”材料提高成功率，但不应让成功率轻易达到 100%。融合要保留赌性和戏剧性。

融合结果例子：

| 素材 | 可能结果 | 效果 |
| --- | --- | --- |
| 火种 + 裂片 | 爆燃碎片 | 爆炸后留下燃烧地面 |
| 冰晶 + 震荡 | 冰封冲击 | 击退被冻结或减速敌人时造成额外伤害 |
| 雷鸣 + 回流 | 雷能回路 | 雷链伤害转化部分主动能量 |
| 磁币 + 赏金 | 悬赏磁场 | 自动吸钱，并提高范围内敌人掉钱 |
| 导线 + 引力 | 牵引导线 | 能源线附近敌人被轻微吸附 |
| 聚焦 + 标记 | 弱点棱镜 | 被标记敌人受到更高暴击伤害 |

## 9. 怪物设计

怪物需要逼迫玩家使用不同圣器和布局，而不是只堆最高 DPS。

### 9.1 基础怪物类型

| 怪物 | 特点 | 克制方式 |
| --- | --- | --- |
| 行尸 | 标准单位，数量中等 | 任意基础输出 |
| 群虫 | 数量多、生命低 | 范围、燃烧、能源线 |
| 甲壳兽 | 高护甲、低速度 | 腐蚀、破甲、雷击 |
| 疾行者 | 高速度、低生命 | 减速、冻结、能源线 |
| 护盾怪 | 有能量盾，盾破前抗异常 | 高频攻击、雷鸣 |
| 分裂体 | 死亡后分裂小怪 | 范围收割、燃烧 |
| 吞能者 | 经过能源线时降低线伤 | 塔本体输出、控制 |
| 掘地怪 | 短暂绕过部分能源线 | 震荡、标记、主动雷电 |

### 9.2 精英怪

精英怪应有清晰的词缀：

- **厚甲**：护甲提高，需要破甲。
- **迅捷**：移动速度提高，需要控制。
- **抗燃**：燃烧时间降低，逼迫换伤害类型。
- **吸能**：经过能源线时恢复少量生命。
- **召唤**：周期性召唤小怪。
- **狂暴**：低血时加速冲向能源塔。

精英怪掉落圣器或融合材料，是局内构筑的重要节点。

### 9.3 Boss

Boss 不应只是大血条，而应测试玩家布局。

Boss 设计例子：

1. **裂山巨像**
   - 周期性获得护甲。
   - 会砸碎一块资源地形，改变资源收益。
   - 破甲流和集中火力有效。

2. **吞线母体**
   - 经过能源线时吸收线伤并获得护盾。
   - 逼迫玩家临时关闭部分远线，或改用塔本体输出。

3. **晶巢女王**
   - 周期性召唤小怪。
   - 小怪死亡产生主动能量，适合技能流爆发。

## 10. 场景物件与开采

场景物件让地图不只是路径和塔位。

| 物件 | 生命值 | 掉落 | 战术意义 |
| --- | --- | --- | --- |
| 石块 | 低 | 金币、少量材料 | 开局补经济 |
| 矿石 | 中 | 材料、主动能量 | 支持精炼和技能 |
| 古代遗迹 | 高 | 圣器胚胎、融合材料 | 中后期目标 |
| 能源水晶 | 中 | 主动能量 | 救场技能资源 |
| 封印山体 | 高 | 稀有圣器或塔位 | 长线投资 |

玩家可以命令基础塔攻击场景物件。建议提供两种操作：

- 点击物件，指定附近塔优先攻击。
- 在塔面板中切换目标优先级：怪物优先、地形优先、低血优先、高甲优先、最近能源塔优先。

首版中场景物件不建议直接改变怪物路径，避免寻路复杂度过高。可以先让它们解锁建造格、掉落资源和提供战术目标。

## 11. 主动技能

主动技能消耗主动能量，由能源塔释放。

| 技能 | 消耗 | 效果 | 使用场景 |
| --- | --- | --- | --- |
| 能源塔超频 | 60 | 8 秒内总功率提高 50% | 怪潮、Boss 爆发 |
| 天降雷电 | 40 | 对目标区域造成高额雷电伤害 | 清精英、补漏 |
| 能源线过载 | 50 | 所有能源线转伤效率提高，并附带短暂减速 | 远线构筑核心爆发 |
| 紧急护盾 | 45 | 能源塔获得临时护盾 | 失误兜底 |
| 全域回收 | 30 | 吸取地图金币和部分掉落物 | 忙于操作或高压波次 |
| 塔体重启 | 35 | 清除附近塔的负面状态，短暂提高触发率 | 对抗干扰型敌人 |

主动技能应该强，但不能替代塔防布局。技能资源主要来自收集型圣器和地图目标，因此玩家想多放技能就必须牺牲一部分即时火力。

## 12. 建造与经济节奏

### 12.1 基础塔价格

建议基础塔价格随数量递增：

```text
cost = 30 + 8 * 已建基础塔数量 + 2 * 已建基础塔数量^2
```

这样前几座塔便宜，后期铺满会明显变贵。

### 12.2 拆除返还

- 战斗中拆除返还 40% 金币。
- 波间拆除返还 70% 金币。
- 圣器自动回到背包。
- 拆除后能源重新分配，其他塔立刻变强。

这鼓励玩家根据波次调整布局，而不是被早期决策锁死。

### 12.3 波次奖励

波次结束奖励建议包含：

- 固定金币。
- 剩余能源塔生命奖励。
- 快速清怪奖励。
- 地形开采进度奖励。
- 可选圣器奖励三选一。

“三选一”可以让构筑更有方向，同时减少纯随机掉落带来的挫败感。

## 13. 局内流程

### 13.1 单局结构

首版建议每局 20 波，约 20 到 30 分钟。

阶段：

| 阶段 | 波次 | 目标 |
| --- | --- | --- |
| 开局 | 1-5 | 学会建塔、供能、基础圣器 |
| 扩张 | 6-10 | 引入地形开采、收集型圣器 |
| 构筑 | 11-15 | 流派成型，出现精英词缀 |
| 压力 | 16-19 | 多路线、高强度混合怪 |
| 决战 | 20 | Boss 检验布局 |

### 13.2 战斗前准备

每波前给玩家短暂准备时间：

- 建造或拆除基础塔。
- 调整圣器。
- 选择塔目标优先级。
- 指定需要攻击的场景物件。
- 查看下一波怪物预告。

### 13.3 战斗中操作

战斗中玩家可以：

- 使用主动技能。
- 指定集火目标。
- 临时切换塔目标优先级。
- 建造新塔，但价格或建造时间略高。
- 拾取或回收掉落物。

战斗中操作不应过于繁琐。收集型圣器和全域回收技能可以减少捡东西压力。

## 14. 流派设计

### 14.1 近塔高效流

特点：

- 大量塔贴近能源塔，减少距离衰减。
- 依靠增幅型圣器叠加塔本体输出。
- 能源线伤害较低，但塔稳定强。

优势：

- 简单直接，适合新手。
- 对吞能者等抗线敌人更稳定。

弱点：

- 覆盖地图范围小。
- 容易被高护甲或多路线拖垮。

### 14.2 远线切割流

特点：

- 故意把部分塔放远，让能源线穿过怪物路径。
- 使用导线、引力、震荡、能源线过载等圣器和技能。

优势：

- 对大量小怪极强。
- 地图掌控感强，视觉表现好。

弱点：

- 远塔本体弱。
- 遇到吞能者、掘地怪时需要补充手段。

### 14.3 元素连锁流

特点：

- 用火、冰、雷、腐蚀等攻击型圣器制造状态联动。
- 需要合理搭配不同塔，而不是只堆同一种。

优势：

- 上限高，适合喜欢构筑的玩家。
- 对精英和 Boss 有针对性。

弱点：

- 成型依赖圣器。
- 操作和理解成本高。

### 14.4 经济滚雪球流

特点：

- 早期投入收集型圣器。
- 快速开采地图资源，获得更多圣器和材料。
- 中后期通过精炼和融合反超。

优势：

- 长局收益高。
- 融合玩法参与度高。

弱点：

- 前期战斗压力大。
- 融合失败会造成明显损失。

### 14.5 主动技能流

特点：

- 用回流、采晶、充能等圣器快速获得主动能量。
- 靠能源塔超频、雷电、能源线过载打关键波次。

优势：

- 爆发强，救场能力强。
- 操作爽感明显。

弱点：

- 技能空窗期较弱。
- 需要准确判断释放时机。

## 15. 用户界面

### 15.1 主界面信息

主界面必须展示：

- 能源塔生命。
- 能源塔等级。
- 当前金币、材料、主动能量。
- 当前总功率。
- 当前基础塔有效功率总和、能源线功率总和。
- 当前供能塔数量。
- 下一波怪物预告。
- 当前选中塔的供能百分比。

### 15.2 建造预览

玩家拖动基础塔到格子上时，应显示：

- 是否在供能范围内。
- 距离能源塔多少格。
- 建造后该塔预计供能百分比。
- 建造后所有塔平均供能变化。
- 能源线将经过的路径。
- 能源线预计伤害。

### 15.3 塔面板

选中基础塔后显示：

- 当前功率。
- 攻击速度、伤害、射程、触发率。
- 已挂圣器。
- 圣器槽位。
- 目标优先级。
- 拆除按钮。
- 暂停供能按钮。

“暂停供能”很重要。玩家可以临时让某座塔断电，使其他塔变强，也可以关闭不利的能源线。

选中能源塔后显示：

- 当前等级和下一等级预览。
- 当前总功率、基础塔有效功率总和、能源线功率总和。
- 当前供能半径、主动能量上限、能源线转伤效率。
- 升级所需金币和材料。
- 升级后所有塔平均供能变化。
- 满级后可选的核心模块。
- 当前已选择的核心模块。

### 15.4 圣器背包

背包需要支持：

- 按类型筛选：攻击、收集、增幅。
- 按品质筛选。
- 查看圣器联动提示。
- 一键装备到选中塔。
- 精炼入口。
- 融合入口。
- 锁定圣器，避免误融合。

## 16. 美术与表现

### 16.1 视觉基调

建议基调：古代遗迹科技 + 能源符文 + 机械炮塔。

关键词：

- 地面符文线路。
- 半透明能源光束。
- 低矮机械塔体。
- 圣器作为悬浮核心或插槽模块。
- 怪物偏异化生物或失控构装体。

### 16.2 能源表现

能源塔：

- 中央有稳定旋转的核心。
- 供能越紧张，核心跳动越急促。
- 超频时核心外圈展开，地面符文亮起。

基础塔：

- 没有圣器时外观朴素。
- 挂载圣器后出现不同颜色和模块。
- 供能不足时发光变弱、发射间隔明显变长。

能源线：

- 贴地流动，不遮挡单位。
- 线损越高越亮。
- 造成伤害时出现细小电弧或灼痕。

### 16.3 读图优先级

2.5D 塔防最怕信息糊在一起。首版表现应遵守：

- 怪物轮廓永远清晰。
- 能源线贴地，透明度适中。
- 掉落物小而亮，但不挡路径。
- 圣器特效要分层，不能盖住血条和路径。

## 17. 音频方向

音频可以强化能源系统：

- 能源塔常驻低频嗡鸣。
- 新塔接入时有短促“接线”音。
- 供能不足时塔发射音变弱。
- 能源线伤害使用电流擦过地面的声音。
- 圣器融合成功使用清脆上扬音，失败使用核心碎裂音。
- 超频时音乐加入更强节奏层。

## 18. 新手引导

第一局教学目标：

1. 让玩家放置第一座基础塔。
2. 展示能源线连接。
3. 让玩家观察第二座塔导致两座塔供能下降。
4. 让玩家把一座塔放远，看到塔变弱但能源线变强。
5. 给予第一个攻击型圣器。
6. 指挥塔攻击石块，获得金币或圣器。
7. 升级一次能源塔，观察总功率和供能变化。
8. 使用一次主动技能。
9. 完成一次精炼。
10. 在安全环境中展示一次融合失败的风险，建议用教学专用材料，不销毁玩家核心圣器。

教学文案应少而直接，把解释放在玩家操作之后。例如玩家建第二座塔后，再提示“所有塔共用能源，新塔会让旧塔分到的功率变少”。

## 19. 首版内容范围

为了尽快验证核心乐趣，首版原型建议控制范围。

### 19.1 必须完成

- 1 张测试地图。
- 1 座能源塔。
- 局内能源塔升级。
- 能源塔满级后的核心模块三选一。
- 基础塔建造、拆除、供能、断电。
- 能源距离衰减。
- 能源线伤害。
- 2 个圣器槽。
- 12 件圣器：
  - 攻击型 5 件：火种、冰晶、雷鸣、裂片、腐蚀。
  - 收集型 3 件：磁币、采晶、回流。
  - 增幅型 4 件：鼓舞、导线、聚焦、充能。
- 圣器精炼。
- 圣器融合与失败销毁。
- 6 种普通怪。
- 2 种精英词缀。
- 1 个 Boss。
- 4 种场景物件。
- 3 个主动技能：能源塔超频、天降雷电、能源线过载。

### 19.2 首版暂缓

- 多能源塔。
- 复杂地形破坏后改变怪物路径。
- 复杂局外数值养成树。
- PVP 或排行榜。
- 大量剧情文本。
- 过多品质和词条。
- 超过 3 个圣器槽位。

这些内容不是不做，而是等核心能源和圣器循环验证成立后再扩展。

## 20. 平衡原则

### 20.1 能源平衡

- 增加一座塔必须有明显代价。
- 远塔必须有价值，但不能让远线成为唯一最优解。
- 能源线强在持续和覆盖，不强在瞬间爆发。
- 暂停供能应成为高级操作，但不能变成每秒都要切换的负担。
- 能源塔升级可以提高全局上限，但不能消除塔数量分流带来的压力。
- 核心模块只在能源塔满级后出现，避免玩家过早锁定单一最优流派。

### 20.2 圣器平衡

- 攻击型提供清怪和打 Boss 的主力。
- 收集型提供长期收益，但短期火力弱。
- 增幅型提高阵地效率，但需要周围有值得增幅的塔。
- 融合产物可以强，但要有构筑条件和失败风险。

### 20.3 经济平衡

- 玩家应该能在不完美操作下通关普通难度。
- 精炼是稳健成长，融合是高风险成长。
- 能源塔升级是稳定成长，但会和建塔、精炼争夺金币和材料。
- 地图开采收益要足够诱人，让玩家愿意分火。
- 波次奖励要保证失败后玩家理解自己差在哪里，而不是觉得随机掉落不公平。

## 21. 数值示例

假设能源塔总功率为 100，供能半径为 7。

### 21.1 只有 2 座塔

| 塔 | 距离 | 理论分功率 | 衰减后有效功率 | 线损功率 |
| --- | --- | --- | --- | --- |
| A | 2 格 | 50 | 44 | 6 |
| B | 6 格 | 50 | 24 | 26 |

结果：

- A 是强近塔。
- B 本体较弱，但能源线伤害明显。
- 总功率仍是 100，其中 68 用于塔，32 用于能源线。

### 21.2 有 5 座塔

| 塔 | 距离 | 理论分功率 | 衰减后有效功率 | 线损功率 |
| --- | --- | --- | --- | --- |
| A | 1 格 | 20 | 19 | 1 |
| B | 2 格 | 20 | 18 | 2 |
| C | 3 格 | 20 | 16 | 4 |
| D | 5 格 | 20 | 12 | 8 |
| E | 7 格 | 20 | 5 | 15 |

结果：

- 单塔都比 2 塔时弱。
- 远塔 E 几乎不适合作为主输出，但它的能源线很有战略价值。
- 玩家可能暂停 E 的供能，让 A-D 短期变强，也可能保留 E 作为地面伤害线。

## 22. 关卡设计示例

### 22.1 第一张地图：断裂矿场

地图结构：

- 能源塔在地图中央偏左。
- 怪物从右上和右下两条路进入，最终汇向能源塔。
- 中央有两块石块，挡住最佳远线位置。
- 左下有能源水晶。
- 右侧有古代遗迹，但攻击它会分散清怪火力。

设计目的：

- 教玩家基础能源线。
- 教玩家远塔可以让线穿过路径。
- 教玩家攻击资源物件。
- 教玩家在双路线压力下使用主动技能。

波次节奏：

| 波次 | 内容 | 教学点 |
| --- | --- | --- |
| 1 | 少量行尸 | 建塔和攻击 |
| 2 | 行尸 + 石块提示 | 攻击地形 |
| 3 | 群虫 | 范围和能源线价值 |
| 4 | 疾行者 | 减速和线切割 |
| 5 | 精英厚甲行尸 | 腐蚀或集中火力 |
| 6-9 | 双路线混合怪 | 布局压力 |
| 10 | 小 Boss 裂山巨像 | 检验破甲和技能 |

## 23. 局外成长建议

局外成长可以提升每局开始时的能源塔初始等级，但不应直接让玩家跳过局内构筑。

能源塔局外等级建议：

| 局外等级 | 每局初始能源塔等级 | 附加开局收益 |
| --- | --- | --- |
| 1 | Lv.1 | 无 |
| 2 | Lv.1 | 初始金币 +20 |
| 3 | Lv.2 | 开局总功率、生命、主动能量上限按 Lv.2 生效 |
| 4 | Lv.2 | 初始材料 +20 |
| 5 | Lv.3 | 开局总功率、生命、主动能量上限按 Lv.3 生效 |

局外初始等级最高建议到 Lv.3。玩家仍然需要在局内把能源塔升到 Lv.5，才能选择核心模块。这样老玩家开局更顺，但不会开局直接拿到满级模块。

正式版本还可以考虑：

- 圣器图鉴：记录见过和融合出的圣器。
- 起始圣器选择：每局开始从已解锁普通圣器中选 1 件。
- 能源塔外观升级：不影响强度，只表现局外成长。
- 难度词缀：更高难度增加怪物词缀和融合奖励。
- 挑战地图：固定圣器池、固定能源塔位置，考验布局。

局外成长应以解锁玩法和目标为主，不应让老玩家因为永久数值过高而失去能源分配的核心压力。

## 24. 技术实现要点

虽然本文件是玩法策划，但以下规则会影响实现复杂度。

1. **能源分配每次建造、拆除、断电、供能范围变化时重算**
   - 不需要每帧重算。
   - 战斗中若有临时增幅，再按持续时间更新。
   - 能源塔升级或选择核心模块后，也需要立即重算供能、能源线和相关 UI。
   - 重算后必须校验 `sum(P_delivered_i) + sum(P_line_i) == P_total`。若存在浮点误差，以最后一条能源线或显示层四舍五入吸收误差，不能让总功率在规则层变多。

2. **能源线使用独立伤害区域**
   - 可以用线段碰撞或格子覆盖实现。
   - 首版若使用格子制，建议把能源线经过的格子缓存为伤害格。

3. **圣器效果采用事件触发**
   - 命中事件、击杀事件、怪物死亡事件、波次结束事件、地形受击事件。
   - 这样攻击、收集、增幅可以复用同一套事件系统。

4. **状态效果统一管理**
   - 燃烧、减速、冻结、腐蚀、标记、击退、护盾等状态需要统一持续时间和叠加规则。

5. **融合结果使用配方表 + 权重池**
   - 同名融合走固定升级池。
   - 跨类融合走权重随机池。
   - 这样既能控制产物，又保留随机惊喜。

6. **能源塔等级使用配置表**
   - 每级总功率、生命、供能半径、主动能量上限、能源线转伤效率都从配置读取。
   - 局外初始等级只决定开局读取哪一行配置。
   - 核心模块作为满级后的一次性局内状态保存，不与局外初始等级绑定。

## 25. 主要风险与解决方向

| 风险 | 表现 | 解决方向 |
| --- | --- | --- |
| 能源公式太难懂 | 玩家不知道为什么塔变弱 | 建造预览、供能百分比、能源塔面板必须清晰 |
| 功率守恒被误解 | 玩家以为能源线伤害加成会凭空增加功率 | UI 分开显示基础塔功率、能源线功率和转伤效率 |
| 远线过强 | 玩家无脑拉最长线 | 加入抗线敌人、线伤递减、远塔本体弱 |
| 塔太多导致操作混乱 | 玩家频繁换圣器和断电 | 限制槽位、提供批量筛选和暂停供能快捷操作 |
| 融合失败挫败感过强 | 玩家失去关键圣器后不想玩 | 提供锁定、稳定剂、三选一奖励和低风险精炼路线 |
| 收集型收益失衡 | 要么必拿，要么没人拿 | 让收集型占火力和供能，但中后期收益明显 |
| 能源塔升级过强 | 玩家靠永久等级碾压，能源压力消失 | 局外初始等级上限控制在 Lv.3，核心模块必须局内满级后选择 |
| 视觉信息过载 | 能源线、炮弹、怪物混在一起 | 降低地面线透明度，怪物血条和轮廓优先 |

## 26. 第一阶段验证目标

第一阶段不追求内容量，重点验证 5 个问题：

1. 玩家是否能理解“总功率恒定，塔越多单塔越弱”。
2. 玩家是否愿意为了能源线伤害把塔放远。
3. 圣器是否真的让同一种基础塔产生不同玩法。
4. 攻击地形是否能形成有趣的火力取舍。
5. 能源塔升级是否能带来稳定成长，但不破坏能源压力。
6. 满级核心模块是否能让本局构筑方向更明确。
7. 精炼和融合是否能支撑一局内的构筑期待。

如果这些问题成立，后续再扩展关卡、圣器、怪物和局外成长。

## 27. 推荐下一步

下一步建议先做一份最小原型需求：

1. 只做 1 张地图和 10 波怪。
2. 只做 6 件圣器：火种、冰晶、雷鸣、磁币、导线、回流。
3. 做能源塔 5 级升级，但只做 3 个核心模块。
4. 只做 2 个主动技能：能源塔超频、能源线过载。
5. 先不做完整美术，用清晰临时图形表现能源线、供能百分比和圣器效果。
6. 原型测试重点放在能源分配手感，而不是数值丰富度。

这款游戏最关键的不是“塔防内容够多”，而是能源系统能不能让玩家每一次建塔都产生一点犹豫和一点期待。只要这个核心成立，圣器、怪物、地图都可以围绕它持续扩展。
